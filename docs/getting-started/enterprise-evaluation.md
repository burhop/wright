# Enterprise Evaluation

Wright is public-alpha software. Enterprise evaluators should review the local-first and bring-your-own-AI boundaries before running a pilot.

## Evaluation Checklist

- Confirm Docker appliance health locally.
- Confirm model/provider endpoint ownership and credential handling.
- Review `SECURITY.md` and the release notes.
- Review MCP validation boundaries before enabling selected engineering tools.
- Contact `wright@makerengineer.com` for support, integration, or pilot questions.

No hosted backend, LLM, proprietary engineering tool, production SLA, or paid support contract is bundled with the public-alpha image.
