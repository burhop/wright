# Hermes Desktop

Hermes Desktop is a native desktop experience. Wright's Docker appliance runs the local Wright API and web UI; it is not a desktop GUI container.

Use one of these paths:

- Run Wright's Docker appliance and open the web UI in a browser.
- Install the Wright Hermes plugin when you already use Hermes Desktop or Hermes CLI.
- Use `/wright status` to verify the bridge once Hermes and Wright are running.

See [Existing Hermes Plugin](hermes-plugin.md) for the plugin install flow.
