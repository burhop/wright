# Launch Indicators

Review these after public alpha announcements and releases:

| Indicator | Source | Cadence | Meaning |
| --- | --- | --- | --- |
| Repository stars/watchers | GitHub | Weekly | Discovery and interest. |
| GitHub Discussions and issues | GitHub | Weekly | Support load and contributor interest. |
| Release downloads | GitHub Releases | Per release | Artifact consumption. |
| Docker pulls | Docker Hub/GHCR | Weekly after first publish | Appliance adoption. |
| PyPI downloads | PyPI stats | Weekly after first publish | Helper-package discovery. |
| Docs visits | GitHub Pages analytics if enabled | Weekly | Install-doc traffic. |
| Sponsorship inquiries | GitHub Sponsors and email | Monthly | Funding traction. |
