# Visibility Checklist

Use this checklist before a public alpha announcement.

- Repository description explains local-first agent orchestration for physical engineering.
- Repository topics include engineering AI, MCP, local-first, CAD/CAE/CAM, Docker, and FastAPI/React where appropriate.
- README first screen shows what Wright is, who it is for, how to try it, and current alpha status.
- Docker Hub and GHCR listings link to docs, issues, security policy, releases, sponsorship, and `wright@makerengineer.com`.
- PyPI listing for `wright-engineering` links to docs, issues, security policy, releases, sponsorship, and `wright@makerengineer.com`.
- Demo gallery has at least three public-alpha demo slots.
- Release notes explain BYO-AI, skipped MCP validation, platform support, and known limits.
- Contribution paths include issues, discussions, MCP validation, docs, and good-first issues.
