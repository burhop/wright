# Partner Brief

Wright is a public-alpha, local-first agent orchestration platform for physical engineering workflows. It helps engineers coordinate AI agents, deterministic engineering tools, and MCP servers while keeping model choice, credentials, local data, and workstation deployment under the operator's control.

## Partner Relevance

- Workstation and edge-AI vendors can show local engineering AI workflows.
- Cloud/API providers can support model evaluation and CI testing.
- Engineering-tool vendors can validate MCP integrations safely.

## Current Requests

- Hardware access for Linux, Windows, and GB10-class validation.
- Cloud/API credits for continuous model testing.
- Technical introductions for engineering tool integrations.
- Co-marketing only after install paths and demos are repeatable.

NVIDIA Inception is deferred until Wright has an eligible company or organization. Dell and other hardware partner outreach can proceed through direct contacts using this brief.
