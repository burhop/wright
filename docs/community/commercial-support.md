Public alpha does not include a production SLA or bundled paid support. Teams can contact `wright@makerengineer.com` to discuss:

- Private setup help.
- MCP integration and validation.
- Offline or local-first deployment planning.
- Hardware/workstation validation.
- Custom engineering workflow automation.

Any paid support, timeline, or deliverable must be agreed separately and should not be implied by public-alpha install docs.
